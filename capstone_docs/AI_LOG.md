AI Usage Log -- Capstone Project
---------------------------------------

### Purpose of This Log

This document provides a transparent record of how AI tools were used throughout the development of this capstone project. It outlines the role AI played in supporting ideation, structuring, coding, and documentation while ensuring that all final decisions, implementations, and understanding remain my own.

* * * * *

### AI Tools Used

| Tool | Purpose |
| --- | --- |
| ChatGPT (OpenAI) | Ideation, planning, code templates, documentation, troubleshooting |
| GitHub Copilot (via VS Code) | In‑line code suggestions, autocompletion, refactoring, generating repetitive code |

* * * * *

### Summary of AI Usage by Phase

#### 1\. Project Ideation & Refinement

-   ChatGPT helped brainstorm multiple capstone ideas, refine the problem statement, and align the project with CISBA requirements (SS, BA, DM, CN).

-   Provided feedback to move from a purely academic analysis toward a practical decision‑support system.

#### 2\. Research & Dataset Selection

-   Guided selection of appropriate public datasets (CDC, Census, County Health Rankings).

-   Helped identify relevant variables (life expectancy, income, education, obesity, smoking).

#### 3\. Project Planning & Architecture

-   Assisted in breaking the project into weekly tasks.

-   Helped design the overall pipeline: Data → Cleaning → Model → Dashboard.

#### 4\. Coding & Development

ChatGPT:

-   Generated starter code for data loading, cleaning, merging on FIPS, training a linear regression model, and building the Streamlit dashboard.

-   Helped troubleshoot specific errors (e.g., FIPS type mismatches, multi‑row headers).

GitHub Copilot (VS Code):

-   Provided inline suggestions while writing Python scripts (`data_loader.py`, `app.py`).

-   Autocompleted repetitive Pandas operations (e.g., `dropna()`, `astype()`).

-   Suggested refactoring for cleaner code structure.

-   Generated docstrings and minor comments automatically.

-   All Copilot suggestions were reviewed, tested, and understood before being kept.

#### 5\. Portfolio & Documentation

-   ChatGPT assisted with drafting the main portfolio `README.md`, evolution folder `README.md`s. 

-   Provided structure for the 5‑page synthesis paper and presentation slides outline.

#### 6\. Synthesis Paper & Presentation

-   Helped connect project components to the four CISBA areas.

-   Suggested slide layout and demo narrative.

* * * * *

### Nature of AI Assistance

-   AI was used as a support tool for guidance, suggestions, templates, and efficiency.

-   All outputs were reviewed, modified where necessary, and fully understood before being included.

-   Final decisions, code structure, model interpretation, and writing are my own.

* * * * *

### Reflection on AI Use

Using both ChatGPT and GitHub Copilot accelerated my workflow in several ways:

-   Reduced time spent on boilerplate code and syntax lookup.
-   Helped me overcome specific technical hurdles more quickly.
-   Provided a second pair of eyes for debugging and edge cases.

At the same time, I remained responsible for:

-   Understanding every line of code.

-   Validating model assumptions.

-   Ensuring ethical and accurate presentation of public health data.

This experience taught me how to be an effective augmented developer using AI to enhance, not replace, my own skills.

* * * * *

AI played a supportive role throughout the capstone process from initial ideation to final documentation. The final project reflects my ability to integrate knowledge across CISBA areas while responsibly leveraging modern AI tools.
